## What it does

Gemini is Google's AI assistant. Rebranded from Bard in December 2023, it became Google's flagship multimodal AI product. Available via web (gemini.google.com), mobile app, Google Workspace (Docs, Sheets, Gmail), and API.

In May 2026, the **Gemini 3.x family** is the active lineup. **Gemini 3.5 Flash** (released May 19, 2026) is the new default — it surpasses Gemini 3.1 Pro on coding benchmarks at 25% lower cost. **Gemini 3.1 Pro** remains the flagship with a 2M-token context window, the largest in production. **Gemini 3 Flash** and **Flash-Lite** cover cost-sensitive workloads. All support multimodal input: text, image, audio, video, and file in the same conversation.

Gemini's biggest strength is **Google ecosystem integration**: native AI inside Workspace, real-time web search with citations, and Imagen/Veo for native image/video generation.

## Pricing

- **Free** — Gemini 3.5 Flash, generous daily limits, Workspace integration
- **Google AI Plus ($7.99/mo)** — Gemini 3.5 Flash + 3.1 Pro daily allotment, image generation
- **Google AI Pro ($19.99/mo)** — Gemini 3.1 Pro with 1M context, Deep Research
- **Google AI Ultra ($99.99/mo)** — 20× limits, 20 TB storage, Gemini Omni video, Flow filmmaking
- **$200/mo top tier** — highest quotas, priority access

## Capabilities

- Multimodal input: text, image, audio, video, file
- 1M-token context window
- Built-in real-time web search
- Google Workspace integration (Docs, Sheets, Gmail, Drive)
- Veo video generation (Ultra)
- Imagen image generation
- Code execution sandbox

## Strengths

- Native Google ecosystem integration — best-in-class Workspace experience
- Generous free tier — no paywall friction
- Real-time web search with citations
- True multimodal: image + audio + video in one context
- Best value at $19.99/mo Pro tier with 1M context and Deep Research

## Weaknesses

- Aggressive safety filters — refuses many creative writing requests
- Daily request caps on 3.1 Pro feel tight for power users
- Behind Claude Opus 4.7 on coding benchmarks (3.5 Flash narrows the gap)
- Context loss in long conversations
- G2 rating 4.4/5 — slightly behind ChatGPT

## Ecosystem

Gemini's ecosystem lives primarily inside **Google Workspace**: native AI helper for Docs, Sheets, Gmail, Drive. No standalone marketplace — value comes from deep integration across Google products.
